Conhecimento

Java

É uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems.

HTML

É uma linguagem de marcação utilizada na construção de páginas na Web.Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

Git & Github

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software. GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git.

Javascript

É uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

CSS

É um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags.

Bootstrap

É um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
